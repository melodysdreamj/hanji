# Hanji brand assets

`hanji-hero.webp` (detailed ink illustration — the master for every rendered
surface: favicons, the auth-screen mark, PWA/apple-touch icons, README hero,
`og-image.jpg`), plus `hanji-mark.webp` / `hanji-mark-seal.webp` (silhouette
variants kept as reserve artwork, currently unused) are the Hanji project's
identity artwork, provided by the project owner. Unlike the source code, they
are **not** covered by the repository's AGPL-3.0 license or Sponsor Banner
Exception — do not reuse them to
represent other projects or products. Derived raster sizes under `web/public/`
are generated from these masters.

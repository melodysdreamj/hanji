# Hanji brand assets

`hanji-hero.webp` is the detailed illustration master. The official product
icon is the detailed Hanji illustration represented by
`hanji-mark-round.png` and its app-ready derivatives under `web/public/`,
including `icon-192.png` and `icon-512.png`. Use that official icon for product
identity surfaces such as workspace/page icons, loading and authentication
screens, PWA/apple-touch icons, README imagery, and `og-image.jpg`.

`web/public/mark-128.png` is reserved for favicon generation and other tiny
browser-chrome favicon variants. Do not use it as a workspace icon, page icon,
loading mark, authentication mark, README product icon, or general brand
illustration. The actual browser links use the generated `favicon-16.png`,
`favicon-32.png`, and `favicon.ico` files.

`hanji-mark.webp` / `hanji-mark-seal.webp` are reserve silhouette artwork and
are currently unused. All Hanji identity artwork was provided by the project
owner. Unlike the source code, it is **not** covered by the repository's
AGPL-3.0 license or Sponsor Banner Exception — do not reuse it to represent
other projects or products.
